# Password Generator

## Description
This is a simple Python mini project that generates a strong random password based on the length entered by the user.

## Features
- User chooses password length.
- Generates a random password.
- Uses uppercase and lowercase letters.
- Includes numbers.
- Includes special characters.

## Technologies Used
- Python
- random module
- string module

## How to Run
1. Install Python.
2. Download the project files.
3. Open terminal or command prompt.
4. Run:

python password_generator.py
